# 🌤️ Weather App

A simple and responsive weather application built with **HTML, CSS, and JavaScript**. The app fetches real-time weather information using the **OpenWeather API** and displays the current weather of any city.

## ✨ Features

- Search weather by city name
- Display temperature in Celsius
- Show humidity
- Display weather description
- Display weather emoji based on weather conditions
- Handle invalid city names gracefully

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/megTechie/weather-app-js.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

4. Enter a city name and click **Get Weather**.
